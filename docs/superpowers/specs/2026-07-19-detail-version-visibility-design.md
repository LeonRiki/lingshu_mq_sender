# 详情页版本号可见性设计

## 目标

测试用例详情页和测试记录详情页不显示应用版本号；列表页继续显示版本号和现有更新入口。

## 方案

利用页面现有的 `case-detail-open` 与 `record-detail-open` 状态类，在 CSS 中仅隐藏顶部栏的版本号宿主 `.app-version`。

## 边界

- 不改变版本检查、更新弹窗或更新来源逻辑。
- 不改动列表页的版本号展示。
- 不修改 README、版本号或服务端接口。

## 验证

- 进入测试用例详情页后，顶部栏不显示版本号。
- 进入测试记录详情页后，顶部栏不显示版本号。
- 返回任一列表页后，版本号恢复显示。
